# Auto-Quantification_of_Cell_Growth_Induced_Pressure
A Python-based bio-image informatics pipeline for characterizing the mechanics of bacterial growth under confinement. This R&amp;D tool automates the detection of microfluidic wall deformations to quantify Growth-Induced Pressure (GIP) in millibars (mbar), bridging the gap between high-resolution microscopy and quantitative mechanobiology.
