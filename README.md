# Automated Quantification of Bacterial Growth-Induced Pressure

This repository contains a Python-based **bio-image informatics pipeline** designed to characterize the mechanics of bacterial growth under confinement. The tool automates the detection of microfluidic wall deformations to quantify **Growth-Induced Pressure (GIP)** in millibars (mbar), bridging the gap between high-resolution microscopy and quantitative mechanobiology.
